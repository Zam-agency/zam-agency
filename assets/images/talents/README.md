# /assets/images/talents/

Dossier prévu pour les photos des talents représentés (célébrités, experts) et des architectes/designers.

Important : aucune photo de talent n'a été ajoutée automatiquement. Les portraits de célébrités sont des œuvres protégées (droit à l'image, droits d'auteur des photographes) — seule ZAM, avec les autorisations réelles et à jour de chaque talent, peut choisir et déposer les bons visuels ici.

Convention de nommage suggérée : `prenom-nom.jpg` (ex. `jeanne-damas.jpg`), format portrait 4:5, export web optimisé (WebP ou JPG compressé).
