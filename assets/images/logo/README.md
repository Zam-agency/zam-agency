# /assets/images/logo/

À placer ici :

- `favicon.ico` (32×32 ou 48×48) — référencé dans le `<head>` de chaque page.
- Le logo ZAM en SVG ou PNG haute définition, si vous préférez une image au wordmark CSS actuel (`Za` + `M` doré, géré directement dans `style.css` sous `.logo-mark`).

Aucune image requise pour que le site fonctionne : le logo actuel est du texte stylé (léger, rapide, net sur tous les écrans).
